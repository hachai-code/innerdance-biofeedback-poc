# docs/research/

Research iteration documents, preserved in order. Do not edit retroactively.

Each file is the full output of one research design session. The `RESEARCH_JOURNAL.md` at the repo root contains a summary of each step with the key decision and pivot.

## Reading order

Start with `step5_refined_approach.md` for the current protocol design.
Read `step2_metrics_hypothesis_v2.md` for literature grounding and the full ML approach evaluation.
Read `step3_sequence_models.md` if you are working on the signal-to-signal coupling models (Phase 1+).

## Why all versions are kept

Each iteration changed the design in a meaningful way. The pivot from relaxation to resilience framing (step 4→5) and the discovery of the RMSSD-during-slow-breathing constraint (step 2) are load-bearing decisions that must remain traceable.
