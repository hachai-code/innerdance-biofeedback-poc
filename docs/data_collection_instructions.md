Here are the instructions for data colelction:

If you already want to collect data, you can use the instructions below which need a screenshot and uploading on the Google drive.
In about one week time we plan to have a basic version for a user interface which will enable you to play the playlist and upload the data without using other systems.

1. Download the ECG og data app: ECG Logger

iOS
https://apps.apple.com/de/app/ecg-logger-for-polar-h10/id1595905348?l=en-GB

Android
https://play.google.com/store/apps/details?id=com.mattimononen.ecglogger&pcampaignid=web_share

2. Download playlist from the provided link
Make sure sure you have enough time: 15 minutes additionally to the playlist duration.

3. Put the Polar H10 chest belt on
Moisture on the electrodes for better conenction

4. In the ECG Logger app, connect to the Polar device

5. You should see data being displayed, if belt correctly placed it would show real time signals

6. Press record, the first time you record for the specific device and specific appe account, you need to pay (7 Euro)

7. It will prompt you for a name, type in: <Participant Name>-<Playlist name>- appended in front of the provided name
E.g. Ahilan-REM-<timestamp>.csv

8. Take a resting break for start (no physical of mental activity) for 5 minutes

9. Start the playlist you choose and take a screenshot of the EEG logger so that the time is  visible.

10. Wait until the playlist finishes, then take another 5 minutes break 

11. Click on the record toggle button in the ECG Logger. 

12. Upload the csv ECG Logger file to this location
<drive>
